# Safora QA Engineering Intern Assignment

## Overview

This repository contains my submission for the Safora QA Engineering Intern assignment. The project includes exploratory manual testing documentation and a Selenium-based UI automation script for the Safora website.

Target Website:
https://safora.se/en/

---

# Project Contents

* `Test Case Document.pdf`

  * Exploratory manual testing report covering homepage, navigation, responsiveness, contact form validation, and feature testing.

* `contact_form_test.py`

  * Selenium automation script for the Safora Contact Us form.

* `requirements.txt`

  * Python dependencies required to run the automation script.

---

# Technologies Used

* Python
* Selenium
* Google Chrome
* Visual Studio Code

---

# Automation Test Scenario

The Selenium automation script performs the following actions:

1. Opens the Safora Contact page
2. Fills required form fields
3. Enters test data
4. Submits the form
5. Verifies form interaction behavior
